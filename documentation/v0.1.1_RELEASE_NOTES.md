# Release v0.1.1: Import & Export function, UI refinements

Welcome to the first production-optimized release of **DocuDrop**! This version transforms the app from a web sandbox experiment into a robust, high-fidelity digital wallet specifically architected for Android deployment.

## 🚀 Key Highlights

### 📤 Native Import & Export
Take control of your data. We have integrated **Capacitor's Filesystem and Share plugins** to allow for seamless vault backups.
- **Export**: Generate a secure JSON backup of your entire document vault and save it to your device, Google Drive, or send it via email using the native Android Share Sheet.
- **Import**: Restore your vault from a previously exported backup file with a single tap.

### 🎴 The "Sticky Squeeze" Carousel
We have completely reimagined the document browsing experience. 
- **Digital Wallet Look**: Documents are now displayed as a stack of overlapping cards, mimicking a physical pass manager.
- **Interactive Scrolling**: As you scroll, cards now dynamically "squeeze" and stack at the top of the viewport, keeping your headers visible while maintaining focus on the content you're browsing.
- **Snap-to-Card**: Enabled scroll snapping for a more tactile, native-app feel.

### 🎨 UI Refinements & Decluttering
- **Streamlined Header**: To maximize screen real estate for your documents, the Theme toggle has been moved into the Settings menu.
- **Refined Card Aesthetic**: Documents now feature a premium "Travel Pass" design with improved typography and high-contrast color coding.
- **Dynamic Versioning**: The app now automatically pulls its version number (`v0.1.1`) directly from the project metadata, displayed clearly in the settings panel.

## ⚙️ Technical Improvements
- **Android Optimization**: Stripped all web-sandbox dynamic imports in favor of static resolution, leading to a smaller APK size and faster initial load times.
- **Storage**: Transitioned to IndexedDB for high-performance, local-first persistence of binary document data.
- **PDF Rendering**: Configured a dedicated worker-thread for PDF parsing via CDN to ensure the UI remains responsive even when opening large documents.

## 🗺️ What's Next?
- **Manual Reordering**: Drag-and-drop support to organize your cards exactly how you want them.
- **Swipe Actions**: Gestures for quick deletion and renaming directly from the carousel view.
- **System Integration**: Registering DocuDrop as a target in the Android "Share" menu to "Drop" files directly into the vault from other apps.

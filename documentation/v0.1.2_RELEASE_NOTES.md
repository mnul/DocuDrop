# Release v0.1.2: Smooth Document Viewing & Pinch-to-Zoom

This release focuses on refining the document interaction experience, bringing native-feeling zoom and pan gestures to your digital vault.

## 🚀 Key Highlights

### 🔍 Native Pinch-to-Zoom
The document viewer now supports high-performance, GPU-accelerated zooming and panning.
- **Buttery Smooth**: Integrated `react-zoom-pan-pinch` for a responsive touch experience.
- **Double-Tap**: Quickly zoom in and out with a standard double-tap gesture.
- **Perfect Fit**: Documents now open with a "Fit to Width" aspect ratio by default, eliminating awkward initial scaling issues.

### 🧹 Interface Cleanup
- **Gesture-Driven**: Removed manual zoom buttons to declutter the viewer and maximize content space.
- **Physics-Based Panning**: Enabled velocity-based panning, allowing you to "flick" through long documents naturally.
- **State Persistence**: Zoom levels are now correctly reset when switching between documents.

## ⚙️ Technical Improvements
- **Hardware Acceleration**: Moved scaling logic from React state to CSS Transforms, significantly reducing CPU load on Android.
- **Dependency Update**: Added `react-zoom-pan-pinch` for specialized gesture handling.

## 🗺️ What's Next?
- **Manual Reordering**: Drag-and-drop support to organize your stack.
- **Swipe Actions**: Swipe-to-delete on the main carousel view.
- **System Sharing**: Ability to "Drop" files into DocuDrop directly from the Android Share menu.

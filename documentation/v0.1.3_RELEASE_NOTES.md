# Release v0.1.3: Document Management & Title Renaming

This release introduces the ability to manage your document titles directly within the app, alongside critical UI stability improvements to ensure a seamless viewing experience.

## 🚀 Key Highlights

### ✏️ Document Renaming
Managing your vault is now easier. You can now rename any document after it has been added.
- **Modal Interface**: A clean, focused popup allows you to edit titles without leaving the document viewer.
- **Instant Feedback**: Changes are instantly reflected in the document header and the main vault overview.

### 🎨 UI & Stability
- **Rendering Fix**: Resolved a critical issue where the document viewer would fail to render when toggling between edit and view modes.
- **Keyboard Support**: Added support for the "Enter" key within the rename modal to confirm changes quickly.

## ⚙️ Technical Improvements
- **State Management**: Optimized how document metadata is updated in IndexedDB to ensure data integrity during rename operations.
- **Conditional Logic**: Refactored the viewer header to handle modal states more reliably, preventing UI crashes.

## 🗺️ What's Next?
- **Manual Reordering**: Drag-and-drop support to organize your stack exactly how you want it.
- **Swipe Actions**: Swipe-to-delete on the main carousel view.
- **System Sharing**: Ability to "Drop" files into DocuDrop directly from the Android Share menu.
